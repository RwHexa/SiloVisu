## Allgemeine Beschreibung: Silo-FüllstandsVisualisierung

Die Silo-Füllstandsvisualisierung ist ein übersichtliches und
benutzerfreundliches PC-Programm zur Darstellung des aktuellen
Füllstands eines Silos. Der Messwert wird in Echtzeit von einer externen
Steuerung (z. B. ESP32-Controller) ermittelt und über eine serielle
USB-Schnittstelle an die Software übertragen.

Auf dem Bildschirm wird der Silo-Füllstand grafisch in Form einer
bildlichen Darstellung angezeigt. Der aktuelle Wert wird zusätzlich in
Prozent ausgegeben und farblich im Silo-Bild hervorgehoben. Die
Darstellung passt sich automatisch an die Bildschirmgröße an und bleibt
stets mittig angeordnet -- ideal für verschiedene Monitore,
PC-Screen/Tablet.

Die Software unterstützt sowohl interne Messwerte (z. B. Testbetrieb
über ein Poti) als auch externe Sensorsignale. Über eine einfache
Auswahl des COM-Ports kann die Verbindung zur angeschlossenen Steuerung
hergestellt oder getrennt werden. Ein klar strukturiertes Bedienfeld
ermöglicht das Aktivieren und Deaktivieren der Messdatenübertragung.

**\[Hauptmerkmale:]{.mark}**

* Echtzeit-Visualisierung des Silo-Füllstands(Sensor)
* Prozentanzeige und farbliche Pegeldarstellung(Balken)
* Automatische zentrierte Darstellung für alle Bildschirmgrößen
* Verbindung über USB/serielle Schnittstelle
* Unterstützung externer Sensoren oder Testbetrieb manuell vorgeben
* Einfache COM-Port-Auswahl
* Bedienerfreundliche Oberfläche
* Nur eine Datei-.exe laden, keine Installation erforderlich !

Die Visualisierung eignet sich ideal für landwirtschaftliche
Anwendungen, industrielle Produktionsumgebungen oder SmartHome, bei
denen der Füllstand eines Silos schnell und klar ersichtlich sein soll.
Auch zur Anzeige von Tanks, Zisternen usw... verwendbar.

Es können auch Schaltungen von Visu zur Steuerung(ESP32)vorgenommen werden.
